# Bolt Kits visual assets

Drop approved Fusion exports for the Bolt Kits page in this folder.

- `bolt-kit-hero.png` — current product image for the first screen (supplied source).
- `bolt-kit-hero.webp` — future wide product composition for the first screen, 2400 px wide or larger.
- `bolt-kit-detail-01.webp` — close technical detail, 1600 px wide or larger.
- `bolt-kit-detail-02.webp` — kit or installed application detail, 1600 px wide or larger.

Use WebP where practical. Keep the original source files outside the public site folder.
