# Brake Rotor image assets

Place the supplied brake rotor design PDF and source image files in this folder.

Keep the original PDF here as the reference source. Derived web assets should use descriptive names such as `brake-rotor-rear.webp` or `brake-rotor-social.png`.

The Brake Rotor catalogue currently uses:

- `LBG-F-KR94.webp`, rendered from `LBG-F-KR94.pdf`, for Front.
- `LBG-R-KR94.webp`, rendered from `LBG-R-KR94.pdf`, for Rear.
