04 / Field Notes

Before
the gate.

01 How should riders choose front vs rear sprocket changes?

Front sprocket changes create a larger gearing shift. Rear changes are better for fine-tuning drive feel, torque delivery, and track response.

02 Can the finder search by make, model, year, and type?

Yes. Those four inputs route the application into the dedicated sprocket finder for fitment support.

03 Can buyers cross-reference an OEM code?

Yes. Cross-reference supports search and quoting by mapping the OEM code through the existing LBG SKU system; it does not replace final fitment verification.

04 Does public SKU guidance expose private geometry?

No. Public product guidance supports riders and dealers while protected geometry remains inside the verification workflow.
