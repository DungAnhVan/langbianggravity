Reference notes

What the cross-reference does and does not do.

Is this a fitment guarantee?

No. It is a replacement reference and cross-reference only. Confirm bike model, year, chain size, and the front or rear application separately.

Why use replacement reference wording?

It keeps the lookup clear for search and quote support while avoiding claims that the page is anything more than a cross-reference.

What if an OEM code is missing?

Send the part number, JT SKU, and LBG SKU through the contact flow so the alias row can be added later.
